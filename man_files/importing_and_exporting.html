Импорт данных


Для загрузки данных в программу выберите меню "Данные"-"Импортировать". В настоящий момент поддерживается загрузка электронных таблиц в форматах: .csv, .xls и .xlsx.
По столбцам - переменные для анализа, по строкам - сами наблюдения. Электронная таблица не должна содержать ничего, кроме данных для загрузки. При загрузке данных из .xls и .xlsx данные должны располашаться только на первом листе.

Экспорт данных


Для сохранения матрицы расстояний и таблицы с метками кластеров используется меню "Данные"-"Экспортировать". Для сохранения графиков используется кнопка сохранения в окне графика.
