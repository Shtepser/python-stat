Кластерный анализ


Кластерный анализ - упорядочивание объектов (наблюдений), о каждом из которых имеется информация по нескольким параметрам (переменным) в сравнительно однородные группы (кластеры). В настоящий момент кластеризация осуществляется на основе иерархического подхода, предполагающего наличие вложенных групп (кластеров различного порядка). Для разделения объектов на кластеры строится матрица близости, определяющая, насколько сильны различия между каждой из пар объектов.

Метод кластеризации

Способ, которым вычисляется расстояние между кластерами


Метод ближнего соседа (single) - расстояние между двумя кластерами определяется расстоянием между двумя наиболее близкими объектами (ближайшими соседями) в различных кластерах. Это правило должно, в известном смысле, нанизывать объекты вместе для формирования кластеров, и результирующие кластеры имеют тенденцию быть представленными длинными "цепочками".
d(u,v)=min(dist(u[i],v[j]))

Метод дальнего соседа (complete) - расстояния между кластерами определяются наибольшим расстоянием между любыми двумя объектами в различных кластерах (т.е. "наиболее удаленными соседями"). Этот метод обычно работает очень хорошо, когда объекты происходят на самом деле из реально различных "рощ". Если же кластеры имеют в некотором роде удлиненную форму или их естественный тип является "цепочечным", то этот метод непригоден.
d(u,v)=max(dist(u[i],v[j]))

Метод средней связи (average) - расстояние между двумя различными кластерами вычисляется как среднее расстояние между всеми парами объектов в них. Метод эффективен, когда объекты в действительности формируют различные "рощи", однако он работает одинаково хорошо и в случаях протяженных ("цепочного" типа) кластеров
d(u,v)=∑ijd(u[i],v[j])(|u|∗|v|)

Метрика кластеризации

Способ расчёта разницы (расстояния) между наблюдениями

Простое Эвклидово расстояние (euclidean) - простое геометрическое расстояние в многомерном пространстве
p(x,y) = sqrt(∑(xi - yi)^2)

Квадрат Эвклидова расстояния (sqeuclidean) - в отличие от простого Эвклидова расстояния, придаёт большие веса удалённым объектам
p(x,y) = ∑(xi - yi)^2

Расстояние городских кварталов (cityblock) - среднее разностей по координатам. В большинстве случаев эта мера расстояния приводит к таким же результатам, как и для обычного расстояния Евклида. Однако отметим, что для этой меры влияние отдельных больших разностей (выбросов) уменьшается (так как они не возводятся в квадрат)
p(x,y) = ∑|xi - yi|
